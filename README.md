### 🔥 Smart Contract Ecosystem 

OctoImpact 2.0 uses **three core smart contracts**:

---

#### 1️⃣ ImpactToken.sol — ERC-20 Token Contract

> 🪙 Represents: **User Effort • Contribution • Verified Actions**

##### 📌 Token Features
- Mint functions  
- Burn functions  
- Reward logic  
- Authentication  
- Metadata  

##### ⚙️ Key Functions

**mintTokens(address user, uint amount)**

- Mints tokens when users complete eco-actions  

**burnTokens(address user, uint amount)**

- Used for:
  - DAO voting stake  
  - Unlocking premium features  

**rewardForAction(string actionType)**

- Maps eco-actions → token rewards  

**balanceOf(address user)**

- Returns real-time balance  

---

#### 2️⃣ EcoBadgeNFT.sol — ERC-721 NFT Badge System

> 🏅 Represents: **Milestones & Achievements**

##### 📌 Badge Types
- Weekly achievements  
- Streaks  
- Leaderboard rankings  
- Major sustainability milestones  

##### 🧾 Metadata Includes
- Badge name  
- Description  
- Rarity  
- Timestamp  
- Blockchain hash  
- User ID  

##### ⚙️ Key Functions

**mintBadge(address user, uint badgeType)**

- Issues a badge when criteria are met.

**getUserBadges(address user)**

- Returns every badge owned by the user.

**badgeMetadata(uint tokenId)**

- Displays complete badge info.

---

#### 3️⃣ ImpactDAO.sol — Governance Contract

> 🗳 Enables **Decentralized Decision-Making**

##### 🧠 DAO Capabilities

- Voting on platform decisions
- Proposal creation
- Token-weighted voting
- Proposal approval/rejection
- System upgrades
- New eco-action categories
  
##### ⚙️ Key Functions

**createProposal(string proposalText)**

- Starts a governance proposal. 

**vote(uint proposalId, bool choice)**

- Users cast a vote:
  - YES
  - NO 
Votes are weighted using tokens.

**tallyVotes(uint proposalId)**

Counts and finalizes results.

#### 🧮 Scoring Formula

```
finalImpactScore =

  + streakBonus 
  + categoryWeight
```

- **streakBonus:**  
  Long-term streak effects

  Some categories have higher climate impact  

- **penaltyFactor:**  
  Used for consumption-heavy weeks
